Frontend schema

This schema describes the content item that is returned from the content store.

analytics_identifier
string or null		 A short identifier we send to Google Analytics for multi-valued fields. This means we avoid the truncated values we would get if we sent the path or slug of eg organisations.

string		 A path only. Query string and/or fragment are not allowed.

string

 The user needs this piece of content meets.

 Related items, can be any page on GOV.UK. Mostly used for mainstream content to power the sidebar. Ordering of the links is determined by the editor in Content Tagger.

 Related items, can be any page on GOV.UK. Overrides ‘more like this’ automatically generated links in the beta navigation.
organisations
 All organisations linked to this content item. This should include lead organisations.

 The organisation that published the original version of the page. Corresponds to the first of the ‘Lead organisations’ in Whitehall for the first edition, and is empty for all other publishing applications.

The links for this item. This is what a publishing application sends in a patch_links call.

Attributes

boolean
links
object
finder
 Powers links from content back to finders the content is surfaced on
lead_organisations
 DEPRECATED: A subset of organisations that should be emphasised in relation to this content item. All organisations specified here should also be part of the organisations array.
mainstream_browse_pages
 Powers the /browse section of the site. These are known as sections in some legacy apps.
meets_user_needs
 The user needs this piece of content meets.
ordered_related_items
 Related items, can be any page on GOV.UK. Mostly used for mainstream content to power the sidebar. Ordering of the links is determined by the editor in Content Tagger.
ordered_related_items_overrides
 Related items, can be any page on GOV.UK. Overrides ‘more like this’ automatically generated links in the beta navigation.

 All organisations linked to this content item. This should include lead organisations.

 The organisation that published the original version of the page. Corresponds to the first of the ‘Lead organisations’ in Whitehall for the first edition, and is empty for all other publishing applications.

 The parent content item.

 A largely deprecated tag currently only used to power email alerts.

 Used for displaying related content on most pages, except for step-by-step and fatality notices. Links and their ordering are determined by the machine learning algorithms.
