Attribute taxonomy_all_ids

Name
taxonomy_all_ids
Description
An object containing the content attribute of the meta tag govuk:taxon_ids in 100 character parts. This is stored like this to get around the 100 character limit on attribute values. Regardless of the length of the meta tag, the object will always contain 5 key/value pairs. If the values are less than 500 characters, unused key/value pairs will be undefined. If the values are longer than 500 characters, the extra characters will be lost.
Value
content attribute of meta tag govuk:taxon_ids
